Developed a mind relaxing and calming prompt based frequency sound generator web application.
## Features Includes:
- Prompt Based
- Generate Frequency sound based on your mood (i.e. how're you feeling).

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
